# Haunted-Horizons
<br>
My First Game Development Project.
Haunted Horizons is a 2D side-scrolling game developed in Unity using C#. The game challenges players to navigate a haunted, night-time environment while evading monsters that spawn from both sides of the screen. Players are rewarded with points for each successful evasion, with a high-score tracker to encourage replayability. The game includes smooth player and monster animations, dynamic camera movement, and intuitive UI elements such as Retry, Home, and Quit buttons. A GameOver window pops up when the player loses, displaying their score and the high score. Developed using Unity, Visual Studio, and C#, the project aims to deliver an engaging experience with responsive controls and immersive gameplay. The expected outcome is to create a fun and challenging game that encourages players to improve their performance through replayability.
